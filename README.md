# 🌍 CodeAlpha AI Translator Pro

A professional, user-friendly Language Translation application developed as part of the CodeAlpha internship program. This tool leverages the `deep-translator` library to provide seamless, real-time translations with a modern GUI.

## 📋 Features
- **Intuitive GUI:** Built with Python's Tkinter for a clean, professional user experience.
- **Dropdown Selection:** Easily select your target language from a pre-configured list (no more guessing language codes!).
- **Copy & Clear:** Built-in productivity features to copy translations to your clipboard or reset the workspace instantly.
- **Robust Error Handling:** Alerts and status indicators for a smooth workflow.

## 🛠️ Built With
- **Python:** Core programming language.
- **Tkinter:** GUI framework for the interface.
- **Deep-Translator:** Stable API library for accurate translations.
- **Pyperclip:** For clipboard integration.
